Hexo搭配Minos的美观博客

hexo是静态博客中比较方便的一种，不需要编程基础，就可以搭建出好看的博客页面；
搭配丰富的插件和主题，让你不花一分钱也能搞出好看的个人小天地

hexo中常见的一些指令

hexo -v hexo 版本
hexo g && hexo s 生成并预览(generate && sever)
hexo d 部署到github.io
